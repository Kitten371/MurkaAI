# MurkaAI

MurkaAI — это развивающаяся интеллектуальная система с акцентом на естественное взаимодействие, выразительность и безопасность.

## Структура проекта

- `UPDATES/` — журнал изменений системы
- `Protecting and authenticating file changes` — политика защиты и верификации изменений

## Основные принципы

- Читаемость и ясность коммуникации
- Поведенческая адаптивность системы
- Защита от несанкционированного воздействия
- Изоляция пользовательских данных

## История развития

- Апрель 2026 — создан первый прототип системы
- Май 2026 — реализовано первое крупное обновление

## Назначение

Проект служит основой для развития MurkaAI и фиксации всех ключевых изменений, связанных с её архитектурой, поведением и безопасностью.
